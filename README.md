📘 DSA-PREPARATION
A curated collection of Data Structures and Algorithms (DSA) problems and solutions designed for interview preparation, competitive programming, and academic practice. This repository serves as a structured roadmap for mastering DSA concepts with clean, well-documented Python implementations.

🚀 Features
Topic-wise organization: Arrays, Strings, Linked Lists, Trees, Graphs, Dynamic Programming, Backtracking, and more.

LeetCode / Interview-style problems: Includes classic problems like N-Queens, Edit Distance, Word Search II, Palindrome Partitioning, etc.

Step-by-step solutions: Each problem is solved with clear explanations and optimal approaches.

Beginner-friendly: Code is modular, readable, and easy to adapt for practice.

Continuous updates: More problems and solutions will be added regularly.

🛠️ Getting Started
Prerequisites
Python 3.8+

Basic knowledge of Git

Clone the repository
bash
git clone https://github.com/parthasarathyc/DSA-PREPARATION.git
cd DSA-PREPARATION
Run a solution
bash
python Arrays/two_sum.py
